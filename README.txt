----------------------------------------
Руководство пользователя MailSimulator
----------------------------------------

Для начала работы нужно запустить файл MailSimulator.java

Значения названий:

Пользователь - абстрактный объект, хранящийся в памяти программы.
 Имеет уникальное имя, inbox, outbox, spam, spamFilter.
 Может отправлять и получать сообщения, просматривать inbox, outbox, spam, а также устанавливать spamFilter.
 Храниться в Хранилище пользователей.

Хранилище пользователей - абстрактный объект, хранящийся в памяти программы.
 Содержит в себе всех созданных пользователей.
 Может получать пользователя по имени и проверять существования пользователя с указанным именем.

Главное меню - этап программы, на котором она может принимать команды. Обозначается знаком ">".

Сообщение - абстрактный объект, хранящийся в памяти программы.
 Имеет пользователя отправителя, пользователя получателя, заголовок, текст.
 Может храниться у любого пользователя в либо inbox, либо outbox, либо spam

Спам-фильтр - абстрактный объект, хранящийся в памяти программы.
 Имеет инструкции для определения сообщения как спам.
 Помогает определять куда должно отправиться сообщение для пользователя получателя, в spam или inbox.

Режим чтения спам-фильтров - этап программы, на котором она может принимать названия спам-фильтров.
 Корректно введенные спам-фильтры устанавливаются для конкретного пользователя после выхода из режима.

=======================================================================================================================

Команды (вводятся после ">"):

"add" - позволяет создавать новых пользователей с любым (почти) именем.
    Далее нужно ввести уникальное (ранее не созданное) непустое имя пользователя.
        При некорректном вводе имени программа сообщает об ошибке и выходит в главное меню.

"list" - Выводит пронумерованный список из имен ранее созданных пользователей.

"send" - позволяет отправить сообщения от одного пользователя другому.
    Далее нужно ввести имена (существующие) пользователя отправителя и пользователя получателя, заголовок и текст.
        При некорректном вводе имени программа сообщает об ошибке и выходит в главное меню.

"inbox" - позволяет просмотреть все сообщения, находящиеся в inbox у конкретного пользователя.
    Далее нужно ввести существующее имя пользователя.
        При некорректном вводе имени программа сообщает об ошибке и выходит в главное меню.

"outbox" - позволяет просмотреть все сообщения, находящиеся в outbox у конкретного пользователя.
     Далее нужно ввести существующее имя пользователя.
         При некорректном вводе имени программа сообщает об ошибке и выходит в главное меню.

"spam" - позволяет просмотреть все сообщения, находящиеся в spam у конкретного пользователя.
    Далее нужно ввести существующее имя пользователя.
        При некорректном вводе имени программа сообщает об ошибке и выходит в главное меню.

"setfilter" - позволяет перейти в режим чтения спам-фильтров для их установки для конкретного пользователя.
    Далее нужно ввести существующее имя пользователя.
        При некорректном вводе имени программа сообщает об ошибке и выходит в главное меню.

При попытке ввода любой другой строки в качестве команды, программа сообщает об ошибке и выходит в главное меню.

=======================================================================================================================

Спам-фильтры (вводятся после "Введите название фильтра:"):

"simple" - спам-фильтр, считающей сообщение спамом,
 если в заголовке или тексте содержится слово "spam", либо "спам" в любом регистре.

"keywords" - спам-фильтр, считающий сообщение спамом,
 если в заголовке или тексте содержится хотя бы одно из ключевых слов в любом регистре.
    Далее вводятся ключевые слова через пробел.
        При некорректном вводе слова программа сообщает об ошибке и выходит в режим чтения спам-фильтров.

"repetition" - спам-фильтр, считающий сообщение спамом,
 если в тексте какое-либо слово в любом регистре встречается чаще, чем заданное число раз.
    Далее вводится максимальное число повторений как целое положительное десятичное число.
       При некорректном вводе числа программа сообщает об ошибке и выходит в режим чтения спам-фильтров.

"sender" - спам-фильтр, считающий сообщение спамом, если пользователь отправитель сообщения имеет заданное имя.
    Далее вводится непустое имя отправителя (оно может быть абсолютно любым).
        При пустом вводе программа сообщает об ошибке и выходит в режим чтения спам-фильтров.

"done" - строка, при вводе которой в качестве спам-фильтра, программа выходит из режима чтения спам-фильтров.
    Все корректно введенные спам-фильтры устанавливаются для указанного пользователя.

При попытке ввода любой другой строки в качестве спам-фильтра, программа сообщает об ошибке и выходит в главное меню.
